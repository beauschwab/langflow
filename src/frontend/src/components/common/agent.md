# components/common/ — Common Reusable Components

## Purpose
General-purpose reusable UI components used throughout the application.

## Key Subdirectories

| Folder | Description |
|--------|-------------|
| `GradientWrapper/` | Gradient background wrapper component. |
| `ImageViewer/` | Image viewing component with zoom/pan. |
| `accordionComponent/` | Accordion/collapsible component with folder variant. |
| `animatedNumbers/` | Animated number counter display. |
| `arrayReaderComponent/` | Array data viewer. |
| `catchAllRoutes/` | Catch-all route handler for 404 pages. |
| `crashErrorComponent/` | Application crash error display. |
| `fetchErrorComponent/` | API fetch error display. |
| `fetchIconComponent/` | Async icon fetcher. |
| `genericIconComponent/` | Generic icon renderer — resolves icon names to components. |
| `horizontalScrollFadeComponent/` | Horizontal scroll container with fade edges. |
| `loadingComponent/` | Loading spinner component. |
| `loadingTextComponent/` | Loading text with animation. |
| `numberReader/` | Number display/formatting component. |
| `objectRender/` | Object/JSON data renderer. |
| `pageLayout/` | Standard page layout wrapper. |
| `paginatorComponent/` | Pagination controls. |
| `renderIconComponent/` | Icon rendering with fallbacks. |
| `sanitizedHTMLWrapper/` | Safe HTML rendering with XSS protection. |
| `shadTooltipComponent/` | Tooltip wrapper using shadcn/ui. |
| `skeletonCardComponent/` | Loading skeleton card placeholder. |
| `storeCardComponent/` | Langflow Store component card display. |
| `stringReaderComponent/` | String display component. |
| `tagsSelectorComponent/` | Tag selection component. |
| `timeoutErrorComponent/` | Timeout error display. |
| `viewTriggers/` | View trigger components (e.g., chat trigger). |
