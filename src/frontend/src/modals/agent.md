# modals/ — Modal Dialog Components

## Purpose
All modal dialog components in the Langflow UI. Modals provide focused UI for tasks like chat/playground (IOModal), API configuration, code editing, flow settings, file management, and more.

## Key Subdirectories

| Folder | Description |
|--------|-------------|
| `IOModal/` | **Primary chat/playground modal** — chat interface, voice assistant, file attachments, I/O field views. |
| `apiModal/` | API configuration modal — shows generated code (Python, cURL, JS) for calling flows via API. |
| `baseModal/` | Base modal wrapper providing consistent layout and behavior. |
| `codeAreaModal/` | Code editor modal for editing Python code in custom components. |
| `editNodeModal/` | Full-screen node/component editor with all parameters. |
| `fileManagerModal/` | File manager — upload, browse, and manage files attached to flows. |
| `newFlowModal/` | New flow creation dialog with template selection. |
| `nodeModal/` | Compact node configuration modal. |
| `promptModal/` | Prompt template editor modal with variable highlighting. |
| `templatesModal/` | Template/starter project browser modal. |
| `confirmationModal/` | Generic confirmation dialog. |
| `deleteConfirmationModal/` | Delete confirmation with warning. |
| `dictAreaModal/` | Dictionary/JSON editor modal. |
| `exportModal/` | Flow export modal. |
| `flowLogsModal/` | Flow execution logs viewer. |
| `flowSettingsModal/` | Flow metadata settings editor. |
| `saveChangesModal/` | Unsaved changes warning dialog. |
| `secretKeyModal/` | API key/secret display modal. |
| `shareModal/` | Flow sharing modal. |
| `tableModal/` | Table data viewer modal. |
| `textAreaModal/` | Text area editor modal. |
| `textModal/` | Text viewer/editor modal. |
| `userManagementModal/` | User management modal (admin). |
| `EmbedModal/` | Embed configuration modal. |
