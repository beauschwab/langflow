# components/outputs/ — Output Components

## Purpose
User-facing output components that display results in the playground.

## Key Files

| File | Description |
|------|-------------|
| `chat.py` | `ChatOutput` component — displays AI responses in the chat interface. Handles message persistence and streaming. |
| `text.py` | `TextOutput` component — displays plain text output. |

## Frontend Integration

- These components render as output displays in the Playground panel.
- `ChatOutput` creates chat bubbles with streaming support.
