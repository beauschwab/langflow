# models/folder/ — Folder Database Model

## Purpose
SQLModel for organizing flows into folders, with pagination support.

## Key Files

| File | Description |
|------|-------------|
| `model.py` | `FolderModel` — SQLModel class for the folders table. |
| `pagination_model.py` | Pagination model for folder listing. |
| `constants.py` | Folder-related constants (e.g., default folder name). |
| `utils.py` | Folder utility functions. |
