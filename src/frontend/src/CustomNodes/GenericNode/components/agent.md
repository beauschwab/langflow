## Purpose
Individual sub-components that make up the GenericNode rendering. Each handles a specific aspect of the node UI.

## Key Subdirectories

| Folder | Description |
|--------|-------------|
| `HandleTooltipComponent/` | Tooltip shown when hovering over connection handles. |
| `NodeDescription/` | Node description display/tooltip. |
| `NodeDialogComponent/` | Full node configuration dialog. |
| `NodeInputField/` | Individual input parameter field rendering. |
| `NodeInputInfo/` | Input field information display. |
| `NodeName/` | Editable node name. |
| `NodeOutputParameter/` | Output parameter display. |
| `NodeOutputfield/` | Output field with connection handle. |
| `NodeStatus/` | Build status indicator (idle/building/success/error). |
| `OutputComponent/` | Output value display component. |
| `RenderInputParameters/` | Renders all input parameters for a node. |
| `handleRenderComponent/` | Connection handle rendering logic. |
| `nodeIcon/` | Component icon rendering. |
| `outputModal/` | Expanded output view in a modal dialog. |
