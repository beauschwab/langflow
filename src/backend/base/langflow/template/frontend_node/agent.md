# template/frontend_node/ — Frontend Node Templates

## Purpose
Generates complete frontend node representations from Python component definitions.

## Key Files

| File | Description |
|------|-------------|
| `base.py` | `FrontendNode` — complete node template including all fields, outputs, metadata, and display properties. |
| `custom_components.py` | Custom component-specific frontend node generation. |
| `constants.py` | Frontend node constants. |
