# parameterRenderComponent/components/ — Input Type Renderers

## Purpose
Individual input type renderer components — each handles a specific backend input type.

## Key Subdirectories

| Folder | Description |
|--------|-------------|
| `inputComponent/` | Text input renderer with variable/secret popover. |
| `dropdownComponent/` | Dropdown selection renderer. |
| `floatComponent/` | Float number input. |
| `intComponent/` | Integer number input. |
| `textAreaComponent/` | Multi-line text area. |
| `codeAreaComponent/` | Code editor input (Python, JSON). |
| `inputFileComponent/` | File upload input. |
| `inputListComponent/` | List of text inputs. |
| `keypairListComponent/` | Key-value pair list editor. |
| `dictComponent/` | Dictionary/JSON object editor. |
| `sliderComponent/` | Numeric slider input. |
| `toggleShadComponent/` | Boolean toggle switch. |
| `multiselectComponent/` | Multi-select dropdown. |
| `promptComponent/` | Prompt template editor with variable detection. |
| `tableComponent/` | AG Grid-based table editor. |
| `TableNodeComponent/` | Table node display component. |
| `linkComponent/` | URL/link input. |
| `tabComponent/` | Tabbed content selector. |
| `inputGlobalComponent/` | Global variable selector input. |
| `refreshParameterComponent/` | Refresh button for dynamic parameters. |
| `textInputComponent/` | Simple text input. |
| `strRenderComponent/` | String rendering component. |
| `copyFieldAreaComponent/` | Copyable text field. |
| `emptyParameterComponent/` | Empty/placeholder parameter. |
| `webhookFieldComponent/` | Webhook URL field. |
