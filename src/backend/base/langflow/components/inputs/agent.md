# components/inputs/ — Input Components

## Purpose
User-facing input components that receive data from users in the playground.

## Key Files

| File | Description |
|------|-------------|
| `chat.py` | `ChatInput` component — the primary chat input interface. Receives user messages. |
| `text.py` | `TextInput` component — simple text input field. |

## Frontend Integration

- These components render as input fields in the Playground panel.
- `ChatInput` creates a chat-style message interface.
