# helpers/ — Helper Functions

## Purpose
Shared helper functions for common operations across the backend — flow management, user operations, folder management, and data manipulation.

## Key Files

| File | Description |
|------|-------------|
| `flow.py` | Flow helper functions — flow listing, loading, and management utilities. |
| `user.py` | User helper functions — user lookup and validation. |
| `folders.py` | Folder helper functions. |
| `data.py` | Data manipulation helpers. |
| `custom.py` | Custom component helper functions. |
| `base_model.py` | Base model helper utilities. |
