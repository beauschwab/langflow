## Purpose
Components for transforming, filtering, parsing, and manipulating data within flows. These are the workhorses for data pipeline operations.

## Key Files

| File | Description |
|------|-------------|
| `combine_text.py` | Combines multiple text inputs into one. |
| `parse_data.py` | Parses Data objects into text representations. |
| `filter_data.py` | Filters data by field values. |
| `filter_data_values.py` | Filters data by specific value criteria. |
| `merge_data.py` | Merges multiple Data objects. |
| `parse_json_data.py` | Parses JSON strings into Data objects. |
| `json_cleaner.py` | Cleans and validates JSON strings. |
| `regex.py` | Regex extraction and replacement. |
| `message_to_data.py` | Converts Message objects to Data objects. |
| `create_data.py` | Creates Data objects from key-value pairs. |
| `extract_key.py` | Extracts specific keys from Data objects. |
| `alter_metadata.py` | Modifies metadata on Data objects. |
| `lambda_filter.py` | Python lambda-based data filtering. |
| `parser.py` | General-purpose parser component. |
| `save_to_file.py` | Saves data to files. |
| `data_to_dataframe.py` | Converts Data to DataFrame. |
| `dataframe_operations.py` | DataFrame manipulation operations. |
| `parse_dataframe.py` | Parses DataFrames. |
| `llm_router.py` | LLM-based routing/classification component. |
