# components/data/ — Data Components

## Purpose
Components for loading, fetching, and creating data from various sources.

## Key Files

| File | Description |
|------|-------------|
| `api_request.py` | API Request component — makes HTTP requests with configurable method, headers, body. Supports cURL import. |
| `csv_to_data.py` | CSV file parser component. |
| `json_to_data.py` | JSON file/string parser component. |
| `file.py` | Generic file loader component. |
| `directory.py` | Directory scanner component — loads files from a directory. |
| `url.py` | URL content fetcher component. |
| `sql_executor.py` | SQL query executor component. |
| `webhook.py` | Webhook receiver component. |
| `s3_bucket_uploader.py` | AWS S3 upload component. |
