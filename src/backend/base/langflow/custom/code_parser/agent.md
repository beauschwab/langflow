# custom/code_parser/ — Python Code Parser

## Purpose
AST-based Python code parser that extracts component class definitions, inputs, outputs, and methods from user-written custom component code.

## Key Files

| File | Description |
|------|-------------|
| `code_parser.py` | `CodeParser` — parses Python source code using the `ast` module to extract class definitions, method signatures, and component metadata. |
