# components/core/ — Core Feature Components

## Purpose
Feature-specific components that implement core Langflow UI functionality — app header, flow toolbar, component parameter rendering, sidebar, card displays, and output viewers.

## Key Subdirectories

| Folder | Description |
|--------|-------------|
| `GlobalVariableModal/` | Modal for managing global variables/secrets. |
| `appHeaderComponent/` | Application header bar with navigation, menus, and user profile. |
| `canvasControlsComponent/` | Flow canvas zoom/pan/fit controls. |
| `cardComponent/` | Flow/component card display with drag support. |
| `cardsWrapComponent/` | Card grid/list wrapper. |
| `chatComponents/` | Chat UI components. |
| `codeTabsComponent/` | Code editor with tabs (for API modal, tweaks). |
| `csvOutputComponent/` | CSV data output viewer. |
| `dataOutputComponent/` | Generic data output viewer. |
| `dateReaderComponent/` | Date display component. |
| `dropdownButtonComponent/` | Button with dropdown menu. |
| `dropdownComponent/` | Dropdown selection component. |
| `editFlowSettingsComponent/` | Flow settings editor (name, description, etc.). |
| `flowToolbarComponent/` | Flow editor toolbar — run, undo, redo, save, and view controls. |
| `folderSidebarComponent/` | Left sidebar with folder navigation for organizing flows. |
| `jsonEditor/` | JSON editor component. |
| `jsonOutputComponent/` | JSON output viewer. |
| `parameterRenderComponent/` | **Core parameter rendering engine** — renders all input field types for component configuration. |
| `pdfViewer/` | PDF document viewer. |
| `sidebarComponent/` | Generic sidebar component. |
| `textOutputComponent/` | Text output viewer. |
