# components/helpers/ — Helper Components

## Purpose
Utility components for common operations within flows — memory management, output parsing, structured output, ID generation.

## Key Files

| File | Description |
|------|-------------|
| `memory.py` | Memory retrieval component — fetches chat history from the session. |
| `store_message.py` | Store message component — persists messages to the database. |
| `structured_output.py` | Structured output component — extracts structured data from LLM responses. |
| `output_parser.py` | Output parser component — parses LLM output into structured formats. |
| `create_list.py` | List creation component. |
| `current_date.py` | Current date/time component. |
| `id_generator.py` | UUID/ID generation component. |
| `batch_run.py` | Batch execution component — runs flows in batch. |
