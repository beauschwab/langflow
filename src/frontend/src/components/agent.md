# components/ — Reusable UI Components

## Purpose
All reusable React components organized into three categories: authorization guards, common/shared components, core feature components, and shadcn/ui primitives.

## Folder Structure

| Folder | Description |
|--------|-------------|
| `authorization/` | Route guard components for authentication and role-based access control. |
| `common/` | General-purpose reusable components — icons, loaders, tooltips, paginator, error displays. |
| `core/` | Feature-specific components — header, sidebar, flow toolbar, parameter renderers, card components, output viewers. |
| `ui/` | shadcn/ui primitive components — buttons, inputs, dialogs, sheets, tabs, etc. |
